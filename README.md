# 🏧 ATM Simulation System
Bu proje, temel bankacılık işlemlerini simüle eden bir **C# Windows Forms** uygulamasıdır.

### ✨ Özellikler
* **Kullanıcı Girişi:** Güvenli giriş simülasyonu.
* **Bakiye Sorgulama:** Mevcut bakiyeyi anlık görüntüleme.
* **Para Çekme/Yatırma:** Hesap bakiyesini güncelleme işlemleri.
* **İşlem Geçmişi:** Yapılan tüm işlemlerin listelenmesi.

### 🛠️ Teknolojiler
* **Dil:** C#
* **Arayüz:** Windows Forms Core
* **IDE:** Visual Studio 2022
